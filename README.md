### 브랜치 소개
1. main : 제품으로 출시될 수 있는 브랜치
2. develop : feature 브랜치들을 병합하기 위해 사용하는 개발 브랜치
3. feature : 기능 개발 브랜치
   - login : 인증 관련(로그인, 회원가입, 소셜로그인 등)
   - painRecord : 통증 기록
   - diseasePrediction : 질환 예측
   - home : 홈 페이지, Tab Bar
   - bodyImage : 3D 신체 이미지 불러오기
   - stat : 통증 통계
   - unityInterlock : iOS와 Unity 연동
5. hotfix : 발생한 버그를 수정하는 브랜치
